# 更新日志

## 2026-03-31

### 本次更新
1. 智能计算的限制判断从“排序比较”改成“长宽高一一对应比较”。
2. 现在按 `cartonL <= maxL`、`cartonW <= maxW`、`cartonH <= maxH` 逐项判断是否满足限制。
3. 智能计算候选搜索范围同步改成按长、宽、高各自上限推导，不再按“最大边”统一推导。
4. 无方案时的排查提示改成与新限制口径一致，优先说明长度、宽度或高度哪个超限。
